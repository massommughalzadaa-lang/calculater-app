
> # calculater-app

this app full two number as an input and perform addition , subtraction and solve it

you have to clone this repositery with

> git clone https://github.com/massommughalzadaa-lang/calculater

## What its does;
  +➕ Addition
  + ➖ Subtraction
  + * Multiplication
  + / Division (round to 2 decimal)
  + Power(**)
  + Remainder (%)

  ---



  ### Concepts I used;
* input() to take user input
* Type casting (floating()) to turn text in to numbers
* Varriable to store value
* Arithmetic operation(+ - * / ** %)
* f-string for clean, formatted output
   
   ---

   
#### How to run;

1. Make sure python is installed
2. download or clone this repositories
3. open a terminal in the project folder and run:
4. type two numbers when asked - and see the magic
